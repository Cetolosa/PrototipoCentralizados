Descripción de la Página (Especificación Funcional - Modelo Oficial)

Esta página permite la gestión completa y centralizada de Centros de Costo (CECOs) y Centros de Distribución (CDs) dentro del sistema Aramark.

Funcionalidades para Centros de Costo (CECOs):
• Agregar registros individualmente (con campos opcionales Atributo y Agrupador).
• Subir y exportar masivamente vía Excel.
• Buscar, editar, eliminar y consultar auditoría completa de cada registro.

Funcionalidades para Centros de Distribución (CDs):
• Agregar centros de distribución que permiten incorporar el o los CDs desde los que se envía la guía.
• Esto habilita la selección exclusiva de guías que provengan del origen seleccionado, facilitando la configuración de reglas de negocio específicas por punto de origen y optimizando la trazabilidad y contabilización de guías.

INSTRUCCIÓN OBLIGATORIA PARA REPLICACIÓN (USAR COMO MODELO OFICIAL):
Esta página debe usarse como modelo oficial en TODOS los cambios y módulos que requieran replicar la estructura. Copiar EXACTAMENTE todo el HTML + CSS + JS (incluyendo descripción colapsable y tema oscuro/claro). Esta es la referencia única y obligatoria para mantener consistencia en todo el sistema Aramark.
