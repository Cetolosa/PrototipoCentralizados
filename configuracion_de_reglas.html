Descripción de la Página

Esta página permite la configuración y gestión de reglas de negocio para la contabilización de guías en el sistema Aramark, como periodos de excepción por inventario o vencimientos. Ofrece KPIs clave (total de reglas, activas, de vencimiento y de excepción), un gráfico de distribución por tipo de regla, herramientas para crear nuevas reglas individualmente o cargar masivamente vía archivos Excel (con integración simulada vía webservice), una barra de búsqueda para filtrar registros, y una tabla interactiva con opciones para editar o eliminar reglas existentes.
