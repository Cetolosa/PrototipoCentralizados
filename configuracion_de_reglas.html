Descripción de la Página (Especificación Funcional - Modelo Oficial)

Esta página permite la configuración y gestión de reglas de negocio para la contabilización de guías en el sistema Aramark.

Tipos de regla disponibles:
• Tiempo: Se indica el número de horas desde la salida del CD.
• Excepción: Intervalo de fechas (Inicio / Fin).
• TiempoCD: Igual que Tiempo, pero aplica solo a un CD de origen específico.
• ExcepciónCD: Igual que Excepción, pero aplica solo a un CD de origen específico.

Los campos Atributo y Agrupador son opcionales. Las reglas se pueden crear individualmente o cargar masivamente vía Excel. La tabla permite editar, eliminar y ver auditoría de cada registro.

INSTRUCCIÓN OBLIGATORIA PARA REPLICACIÓN (USAR COMO MODELO OFICIAL):
Esta página debe usarse como modelo oficial en TODOS los cambios y módulos que requieran replicar la estructura. Copiar EXACTAMENTE todo el HTML + CSS + JS (incluyendo tema oscuro/claro, descripción colapsable y comentarios). Esta es la referencia única y obligatoria para mantener consistencia en todo el sistema Aramark.
