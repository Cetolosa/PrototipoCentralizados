Descripción de la Página (Especificación Funcional - Modelo Oficial)

El dashboard permite monitorear en tiempo real el proceso de contabilización automática de las entradas de mercancía asociadas a los contratos. Proporciona visibilidad sobre el estado de ejecución de los BOTs, las reglas de negocio aplicadas y los resultados de cada transacción, facilitando la detección de incidencias, validaciones pendientes o anomalías en el flujo de contabilización.

INSTRUCCIÓN OBLIGATORIA PARA REPLICACIÓN (USAR COMO MODELO OFICIAL):
Esta página debe usarse como modelo oficial en TODOS los cambios y módulos que requieran replicar la estructura. Copiar EXACTAMENTE todo el HTML + CSS + JS (incluyendo tema oscuro/claro, gráficos y KPIs). No modificar la estructura base bajo ningún concepto. Esta es la referencia única y obligatoria para mantener consistencia en todo el sistema Aramark.
