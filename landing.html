Descripción de la Página (Especificación Funcional - Modelo Oficial)

Esta es la página principal de aterrizaje (landing page) del Portal Operativo Aramark, diseñada como el hub central desde donde los usuarios autenticados pueden acceder tanto a las funcionalidades de operaciones generales (como contabilización de guías y registro masivo de inventarios) como a las herramientas de reportabilidad. La página verifica el perfil del usuario almacenado en localStorage (admin, operador o reporter) y muestra solo las secciones relevantes según los permisos, redirigiendo a index.html si no hay sesión activa. Incluye una barra de navegación superior fija, una sección hero con imagen de fondo y overlay para resaltar el título y descripción, un grid de tarjetas de categorías con efectos hover (incluyendo sub-botones que aparecen al pasar el mouse), información del usuario en el centro, y un footer fijo. Además, incorpora funcionalidad de scroll automático suave al hover sobre tarjetas parcialmente visibles para mejorar la usabilidad en pantallas con scroll.

INSTRUCCIÓN OBLIGATORIA PARA REPLICACIÓN (USAR COMO MODELO OFICIAL):
Esta página debe usarse como modelo oficial en TODOS los cambios y módulos que requieran replicar la estructura. Copiar EXACTAMENTE todo el HTML + CSS + JS (incluyendo tema oscuro/claro, descripción colapsable y comentarios). No modificar la estructura base bajo ningún concepto. Esta es la referencia única y obligatoria para mantener consistencia en todo el sistema Aramark.
