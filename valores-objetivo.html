Descripción de la Página (Especificación Funcional - Modelo Oficial)

Objetivo de la página:
Esta página permite definir y gestionar Valores Objetivo y Umbrales (%) dentro del módulo Registro Masivo de Inventarios de Aramark.

Funcionalidades:
• Definir valores por CECO específico, Atributo compartido o Agrupador compartido.
• Validaciones estrictas en tiempo real.
• Tabla dinámica editable con los 4 ejemplos reales.
• Botones Editar, Eliminar y Auditoría en cada fila.
• Exportación a Excel con expansión automática por cada CECO.
• Tema claro/oscuro y descripción colapsable.

INSTRUCCIÓN OBLIGATORIA PARA REPLICACIÓN (USAR COMO MODELO OFICIAL):
Esta página debe usarse como modelo oficial en TODOS los cambios y módulos que requieran replicar la estructura. Copiar EXACTAMENTE todo el HTML + CSS + JS (incluyendo tema oscuro/claro, descripción colapsable, botones por registro y exportación expandida). Esta es la referencia única y obligatoria para mantener consistencia en todo el sistema Aramark.
